# 🍕 Pizza Sales Data Analysis (Excel Project)

## 📌 Overview
This project analyzes a pizza sales dataset using Microsoft Excel. The goal is to extract meaningful insights, identify trends, and support data-driven decision-making.


## 📊 Objectives
- Clean and prepare raw sales data
- Analyze order patterns and customer behavior
- Identify top and bottom performing pizzas
- Calculate key metrics like average quantity per order
- Build an interactive dashboard using Pivot Tables


## 🧰 Tools Used
- Microsoft Excel
  - Pivot Tables
  - Data Cleaning Functions (TRIM, PROPER)
  - Sorting & Filtering
  - Calculated Fields
- Basic Data Analysis Techniques


## 📈 Key Analysis Performed

### 1. Data Cleaning
- Removed extra spaces using `TRIM()`
- Standardized text formatting with `PROPER()`
- Fixed time formatting issues


### 2. KPI Metrics
- Total Revenue
- Total Orders
- Total Quantity Sold
- Average Quantity per Order


### 3. Time-Based Analysis
- Sales by Hour (AM/PM format applied)
- Peak order times identified


### 4. Product Performance
- Top 10 Best-Selling Pizzas
- Bottom 5 Least-Selling Pizzas
- Sales distribution by pizza category


## 📊 Dashboard Features
- Interactive Pivot Tables
- Top/Bottom Filters
- Hourly Sales Breakdown
- Clean and structured layout


## 🔍 Key Insights
- Certain hours drive the highest sales (peak periods)
- A small number of pizza types generate most revenue
- Some products consistently underperform and may need review

## 📁 Files Included
- `pizza_sales_dataset.xlsx` → Raw dataset
- `cleaned_data.xlsx` → Cleaned version
- `dashboard.xlsx` → Final dashboard


## 🚀 How to Use
1. Download the Excel file
2. Open in Microsoft Excel
3. Navigate to Pivot Tables or Dashboard sheet
4. Use filters to explore insights
